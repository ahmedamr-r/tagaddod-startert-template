# TopBar Component

The TopBar component is a comprehensive top navigation bar with TAGADDOD logo and highly customizable dropdown functionality, designed for modern web applications with multi-platform support.

## Overview

TopBar provides a complete navigation header solution with mobile-optimized hamburger menu, warehouse selector dropdown, and extensive customization options. It integrates seamlessly with the Sidebar component for complete layout solutions.

## Import

```jsx
import { TopBar } from '@tagaddod/react';
```

### RTL Best Practices

1. **Always use ThemeProvider** for proper RTL support and automatic font switching
2. **Choose directional icons carefully** - use appropriate arrows for navigation actions
3. **Test dropdown positioning** - ensure popovers align correctly in RTL layouts
4. **Verify mobile menu behavior** - hamburger menu should slide from the correct side
5. **Use proper warehouse names** - ensure Arabic warehouse names display correctly
6. **Consider text length** - Arabic text may be longer/shorter than English equivalents
7. **Test logo positioning** - ensure logo doesn't interfere with RTL text flow
8. **Check custom listbox options** - verify all custom icons and text work in RTL
9. **Validate logout text** - ensure logout appears in the correct language and position
